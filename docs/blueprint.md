# **App Name**: Rainwater Convention 2025

## Core Features:

- Home Page: Display information about the Rainwater Convention 2025, including a hero section, event details, speaker information, and rainwater harvesting info.
- FAQs Page: Provide answers to frequently asked questions about the event using a simple, clean accordion UI.
- Registration Page: Allow users to register for the event via a form. Validates the email address and store form results temporarily in localStorage as a placeholder storage method. On success, a success message will appear.
- Admin Login: A simple login card that lets administrators access a dashboard with restricted credentials
- Admin Dashboard: Display a table listing registered participants fetched from local storage. The dashboard includes a search bar and optional delete button.

## Style Guidelines:

- Primary color: Rain-blue (#0A4D68) to represent water and professionalism.
- Background color: Soft white (#F5F9FA) for a clean, modern look, allowing the content to stand out.
- Accent color: Leaf-green (#0A8B70) to highlight important elements and reinforce the eco-theme. Provides visual contrast, drawing attention to calls to action and interactive components.
- Body and headline font: 'Inter', sans-serif, for a modern, neutral look.
- Use simple, minimalist icons related to water, sustainability, and events.
- Ensure the layout is full-screen with consistent spacing and padding across all pages. Utilize Tailwind CSS grid and flexbox for a fully responsive design on mobile, tablet, and desktop.
- Apply light hover transitions and smooth interactions to improve the user experience without being distracting.
